# HowBIM 단위 변환기

이 저장소는 HowBIM 단위 변환기의 릴리즈를 관리하는 공개 저장소입니다.

## 다운로드

최신 버전은 [릴리즈 페이지](https://github.com/DrgnPark/unit_converter_release/releases/latest)에서 다운로드할 수 있습니다.

## 주요 기능

- 길이, 면적, 부피, 무게 단위 변환
- 미터법과 임페리얼 단위 간 변환
- 사용자 친화적 인터페이스
- 자동 업데이트 기능
